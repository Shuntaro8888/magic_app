# Magic_appです。
Railsチュートリアルで作成したSample_appを魔法使い系SNSに拡張しました。

プロフィールアイコンを魔法使いor魔法動物（画像生成AIによる）に、ユーザー名をFaker gemによってハリーポッターのキャラクター名にしています。
ポストを投稿すると、Gemini-APIによってハリーポッター風な文章に自動変換されます。
